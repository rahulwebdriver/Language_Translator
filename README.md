# Language_Translator

Rahul’s GPT is a Streamlit-based AI application that allows users to input any text and receive a clear explanation in their preferred language. The app uses a large language model via Groq to generate accurate and easy-to-understand responses.

Users can select from multiple languages such as English, Hindi, German, French, Sanskrit, Italian, and Gujarati. The application is designed to simplify complex text and make information more accessible across different languages.
